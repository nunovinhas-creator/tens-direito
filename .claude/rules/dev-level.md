# Nível de Desenvolvimento — Tens Direito

## Nível: EXPERT

Assume conhecimento completo de:
- Pipeline diário (`pipeline-diario.yml`) e os seus 8 steps
- Arquitectura estática: GitHub Pages, branch main, raiz /
- Stack: HTML puro, Python, feedparser, Playwright, BeautifulSoup
- Guardrail de ficheiros protegidos (dois pontos de aplicação)
- Estrutura de dados: `data/scraped/`, `data/mudancas.json`, `data/divergencias.json`

## Estilo de Resposta

- **Língua**: Português de Portugal — sem PT-BR, sem anglicismos evitáveis
- **Respostas**: directas e curtas — sem preâmbulos, sem resumos redundantes
- **Código**: sem comentários óbvios; só comentar o "porquê" quando não-óbvio
- **Sem repetição** de regras já documentadas no `CLAUDE.md`
- **Sem confirmações** do tipo "a tarefa está concluída" — o diff fala por si

## Fluxo de Trabalho

- Editar → testar localmente se possível → commit → push para main
- Um commit por tarefa lógica, mensagem descritiva em PT
- Checklist do `CLAUDE.md` aplica-se a qualquer HTML novo ou modificado
- Não criar ficheiros de documentação extras salvo pedido explícito
