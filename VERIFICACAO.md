# Sistema de Verificação — Tens Direito

## Ciclos de revisão obrigatórios

| Frequência | O que verificar | Quem dispara |
|---|---|---|
| Diário | GitHub Action RSS — nova legislação ou notícia relevante | Automático |
| Semanal | Links de todas as páginas publicadas (404 checker) | /scan |
| Mensal | Valores e escalões das páginas activas | Manual |
| Anual (janeiro) | Tudo — novo IAS, novos escalões, novo OE | Manual |
| Após OE aprovado | Todas as páginas afectadas pelas mudanças | Manual |

## Eventos que disparam revisão imediata
- Publicação de nova Portaria/Despacho no DRE sobre tema coberto
- Notícia de alteração de valor ou condição de acesso
- Comentário de leitor a apontar erro
- GitHub Action a sinalizar ⚠️ REVER

## Registo de erros e correcções
Ficheiro: ERROS-CORRECÇÕES.md
Formato:
- Data de detecção
- Página afectada
- Erro encontrado
- Correcção aplicada
- Fonte que confirmou a correcção
- Data de publicação da correcção

## Política pública de correcções
Cada página tem no rodapé: "Encontraste um erro?
Avisa-nos em contacto@tensdireito.com — corrigimos em 24h."
Esta política é o nosso maior activo de credibilidade.
